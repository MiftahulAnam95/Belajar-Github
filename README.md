# GitHub Beginner Lab

Website belajar GitHub untuk pemula yang ingin melanjutkan pembelajaran HTML, CSS, JavaScript, PHP, dan Laravel ke workflow profesional.

## Fitur

- 20 materi bertahap
- Halaman detail terpisah untuk setiap materi di folder `materi/`
- Quiz interaktif
- Latihan debugging Git dan GitHub
- Recall konsep
- Simulator command, README, dan GitHub Pages
- Mini project portfolio
- Progress tracker dengan `localStorage`
- Dark mode

## Cara menjalankan

Buka `index.html` langsung di browser, atau publish folder ini ke GitHub Pages. Daftar materi ada di `materi.html`, sedangkan isi setiap materi ada di folder `materi/`.
